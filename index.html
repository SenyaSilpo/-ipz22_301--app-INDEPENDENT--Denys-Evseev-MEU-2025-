1.1 Що таке HTML?

HTML (HyperText Markup Language) – це стандартна розмітка веб-сторінок, що використовується для створення та оформлення контенту на сайтах. HTML описує структуру веб-документів, використовуючи теги.

1.2 HTML Редактори коду

Для написання HTML коду використовують редактори тексту або IDE (інтегровані середовища розробки). Популярні редактори включають:

  • Visual Studio Code
  • Notepad++
  • Sublime Text
  • Atom

1.3 HTML Основні приклади

Ось простий приклад HTML документа:

                
                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Моя перша сторінка</title>
                    </head>
                    <body>
                        <h1>Привіт, світ!</h1>
                    </body>
                    </html>

1.4 HTML Елементи

HTML елементи складаються з тегів, які можуть містити текст, зображення та інші медіа. Наприклад:

                
                    <p>Це параграф.</p>
                    <h1>Це заголовок.</h1>

1.5 HTML Атрибути

Атрибути надають додаткову інформацію про HTML елементи. Вони розміщуються всередині тега:

                
                    <a href="https://example.com">Це посилання</a>

1.6 HTML Параграфи

Параграфи в HTML створюються за допомогою тегу <p>:

                
                    <p>Це приклад параграфа.</p>

1.7 HTML Посилання

Посилання створюються за допомогою тегу <a>:

                
                    <a href="https://example.com">Відвідайте наш сайт</a>

Ось робочий приклад посилання:

Відвідайте Wikipedia

1.8 HTML Зображення

Зображення вставляються за допомогою тегу <img>:

                
                    <img src="image.jpg" alt="Опис зображення">

1.9 HTML Таблиці

Таблиці в HTML створюються за допомогою тегів <table>, <tr>, <td>:

                
                    <table>
                        <tr>
                            <th>Заголовок 1</th>
                            <th>Заголовок 2</th>
                        </tr>
                        <tr>
                            <td>Дані 1</td>
                            <td>Дані 2</td>
                        </tr>
                    </table>
Приклад таблиці
Заголовок 1 Заголовок 2
Дані 1 Дані 2
Дані 3 Дані 4
Дані 5 Дані 6

1.10 HTML Списки

Списки можуть бути упорядкованими та невпорядкованими:

Упорядкований список:

                
                    <ol>
                        <li>Перше елемент</li>
                        <li>Друге елемент</li>
                    </ol>
  1. Перше елемент
  2. Друге елемент
  3. Третє елемент

Невпорядкований список:

                
                    <ul>
                        <li>Перше елемент</li>
                        <li>Друге елемент</li>
                    </ul>
  • Перше елемент
  • Друге елемент
  • Третє елемент
